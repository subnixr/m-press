# Install


```sh
git clone https://github.com/subnixr/m-press.git && cd m-press
rm -r .git && git init
pyhton3 -m venv .venv
activate venv/bin/activate
pip install <requirements.txt
npm install
```

# Run

```sh
activate venv/bin/activate
npm run watch
```

# Import/export strings

```sh
npm run csvt:update
```

```sh
npm run csvt:export
```
